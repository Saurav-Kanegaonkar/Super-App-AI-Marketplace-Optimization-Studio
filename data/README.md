# Data Sources

Synthetic source-style data for super app AI marketplace optimization, mobility demand, delivery reliability, and payment adoption.

- `entities.csv`: 36 marketplace domain records.
- `daily_metrics.csv`: 5,040 marketplace domain-day metric rows.
- `source_events.csv`: 760 operational events, QA checks, experiments, and stakeholder asks.
- `recommended_actions.csv`: 220 candidate actions with effort and expected lift.
