# Data Sources

Synthetic source-style data for an AI product roadmap studio in a multi-service super app marketplace.

- `ai_opportunities.csv`: scored AI opportunities across marketplace balance, delivery reliability, wallet adoption, catalog quality, partner earnings, B2B planning, trust operations, and cross-service growth.
- `prd_cards.csv`: product brief content for each opportunity.
- `experiment_plan.csv`: launch test design and KPI guardrails.
- `model_readiness.csv`: event instrumentation, label quality, latency, human review, and fairness checks.

The data is generated by `scripts/score_operating_data.py`. It is synthetic, documented, and intended for portfolio demonstration only.
